Summary:Side panel for Android
Description:
Run a daemon that adds swipe panels to each side of your Android device screen.
The panels contain the currently running applications, so that you can easily
keep track of them and enter any of them at any time easily.

This app is in beta stage; it is not intended for daily use just yet.
