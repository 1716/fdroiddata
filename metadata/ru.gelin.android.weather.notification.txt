Summary:Weather info in notification bar
Description:
Simple application which displays the air temperature and other weather conditions in the notification bar.
The air temperature is always visible like a system clock. Install the skins for a different look: two of
them are available in F-droid. This app now uses the openweathermap API after Google pull the shutters on
its own.
