Summary:OpenStreetMap navigation & routing
Description:
Osmand is a navigation application with access to a wide source of popular open street map data. All map data including vector and raster maps can be stored on the phone memory card for offline usage. Map data packages for many territories are available from Osmand website as well as a desktop application for creating your own.

Limitations of Osmand compared to Osmand+:
- number of map downloads
- no access to Wikipedia offline POIs
