Description:
Mobile version of the Mozilla Firefox Mobile web browser.  Uses the Gecko layout engine to render web pages, which implements current and anticipated web standards. Features include bookmark sync and custom search engines. Since v13 the interface has been changed to speed things up, but any old addons will not work. Please see the website for the list of supported devices.
