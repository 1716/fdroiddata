Summary:Twitter & Identi.ca client
Description:
Twitter and Identi.ca client, with many powerful features. Only basic authentication works for 
Statusnet/Identi.ca. 

If you want a StatusNet/Identi.ca account press the button at the top of the account setup page. Set an 
appropriate REST API url, which is https://identi.ca/api in the case of Identi.ca. Choose basic authentication.
The search API isn't confirmed to be working, so searches will go to Twitter instead. For general info about 
the StatusNet API see http://status.net/wiki/Twitter-compatible_API. You can donate via PayPal here but an 
Alipay donation address is given on the website.
